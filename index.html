🔒 سیاست قفل سورس

  • مخزن عمومی فقط شامل معرفی، مجوز اختصاصی و دموی UI است
  • کد Python / PySide6 و منطق کسب‌وکار در مخزن Private است
  • هیچ فایل سورس، دیتابیس واقعی یا Credential منتشر نمی‌شود
  • بازتولید یا مهندسی معکوس بدون مجوز کتبی ممنوع است

🖥️ استقرار واقعی

  • اجرا روی Windows به‌صورت پرتابل (بدون نصب Python)
  • دیتابیس SQLite محلی — مناسب دفاتر بدون سرور
  • نقش‌ها: مدیر IT و انباردار با دسترسی جدا
  • بسته نصب/پرتابل فقط از طریق مالک نرم‌افزار

معماری سطح‌بالا

جزئیات پیاده‌سازی عمداً عمومی نیست — فقط لایه‌های منطقی

  • Desktop UI — رابط مدرن Enterprise با ناوبری سایدبار و صفحات ماژولار
  • Application Services — احراز هویت، گزارش، چاپ، بکاپ
  • Data Layer — مخازن داده روی SQLite آفلاین
  • Distribution — بسته‌بندی اجرایی برای استقرار در اداره
