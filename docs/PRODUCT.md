# معرفی محصول — سامانه انبارداری هلال احمر

## مسئله

انبار امدادی نیاز به ثبت دقیق ورود/خروج، پیگیری امانت، کنترل دپو و گزارش‌های رسمی دارد؛ اغلب بدون زیرساخت سرور یا اینترنت پایدار در محل.

## راه‌حل

نرم‌افزار **دسکتاپ آفلاین** با رابط راست‌چین فارسی، دیتابیس محلی، چاپ PDF و خروجی Excel — مناسب استقرار سریع در اداره.

## ماژول‌ها

| ماژول | کاربرد |
|--------|--------|
| داشبورد | آمار، نمودار، هشدار کمبود و سررسید |
| اقلام | کاتالوگ کالا و موجودی |
| انبارها | تعریف چند انبار |
| دپو | قفل و انتقال |
| ورودی | ثبت ورود + پیوست |
| خروج | حواله + چاپ |
| امانت | چرخه امانت و بازگشت |
| لیست‌ها | مرور یکپارچه تراکنش‌ها |
| گزارش‌ها | Excel / PDF |
| تنظیمات | تم، آستانه، بکاپ، کاربران |

## کاربران

- **مدیر IT:** تنظیمات، کاربران، بکاپ
- **انباردار:** عملیات روزمره ورود/خروج/امانت

## آنچه در دموی عمومی نیست

- منطق واقعی اعتبارسنجی و تراکنش
- schema دیتابیس
- کد UI واقعی PySide6
- داده‌های عملیاتی سازمان

این سند فقط برای معرفی است.
